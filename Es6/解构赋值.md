## 变量的解构赋值

### 简介

1. ES6允许按照一定模式从数组和对象中提取值，对变量进行赋值，这被称为解构赋值。

2. ```js
   // 数组的结构赋值
   const arr = ['zhang','li','wang']
   let [zhang,li,wang] = arr
   
   // 对象的解构赋值
   const lin = {
        name: '告五人',
        tags:['唯一'，'爱人错过']
   }
   let {name,tags} = lin
   
   // 复杂解构
   
   let caodong = {
       name:'草东没有派对',
       age:21,
       songs:['大风吹','山海','烂泥'],
       gushou:[
           {name:'草动'},
           {name:'鼓鼓'},
           {name:'xx'}
       ]
   }
   
   let {songs:[o,t,s],gushou:[x,y,z]} = caodong
   
   
   ```
