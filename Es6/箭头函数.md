## 箭头函数

###  注意点

1. 箭头函数this指向声明时所在作用域下this的值
2. 箭头函数不能作为构造函数实例化
3. 箭头函数没有arguments
4. 省略写法

###  示例

```js
let fn = ()=>{
     console.log(this)  // windows
}
let school = {
     name:'冬弥',
     getName(){
        let f = ()=> {
             console.log(this)  // school
        }
        f()
     }
}
```

### 提醒

***\*箭头函数不会更改this指向，所以非常适合设置与this无关的回调，比如数组回调、定时器回调，不适合事件回调与对象方法。\****
