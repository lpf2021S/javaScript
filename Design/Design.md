## Web Design Rules

### Shadows

```
1. You don`t hava to use shadows!Only use them if it makes sense for the website personality
2. Use shadows in small doses:don't add shadows to every element!
3. Go Light on shadows, don't make them too dark
4. Use small shadows for smaller elements that should stand out (to draw attention)
5. Use medium-sized shadows for larger areas that should stand out a bit more
6. User large shadows for elements that should really float above the interface
7. Experiment with changing shadows on mouse interaction (click and hover)
8. Bonus: Experiment with glows (colored shadouw）
```

```
1. 只有当它们对网站的个性有意义的时候才使用
2. 不要给每个元素都添加阴影！
3. 把阴影调亮，不要让它们太暗
4. 使用小阴影的较小的元素应脱颖而出(吸引注意力)
5. 在较大的区域使用中等大小的阴影应该更突出一点
6. 用户应该真正浮动在界面之上的元素的大阴影
7. 尝试改变鼠标交互的阴影(点击和悬停)
8. 按钮发光效果
```

### Where TO USE WHITESPACE

```
* 在sections之间使用大量的空白
* 在一组elements之间使用大量的空白
* 在element之间使用空白
* 在groups of elements内部，尽量使用空白代替行
* 一些元素越是在一起，它们就应该越接近!
* start with a lot of whitespace, maybe even to   much! then remove whitespace from there 
*从大量的空白开始，甚至可能到很多!然后删除其中的空格
* Match other design choices,if you have big text or big icons,you need more whitespace.
* 与其他设计选择相匹配，如果你有大的文本或大的图标，你需要更多的空白。
*尝试一个硬规则，例如所有间距都是16px的倍数。
```

#### ICONS

```
* 使用好的icon pack（大量免费和付费的icon pack）
* 只使用一个icon pack，不要混合来自不同icon pack的图标
* 使用 SVG 或者 icon fonts.不要使用 bitmap image(.jpg and.png)
* 调整网站个性!圆度，重量和填充/轮廓(取决于版式)
* 使用图标为文本提供视觉辅助
* 使用图标作为要点
* 使用与操作相关联的图标，并对其进行标记
* 要保持图标的中性，请使用与文本相同的颜色。
  为了吸引更多的注意，使用不同的颜色
* 不要让图标比它们的设计用途更大。如果需要，用一个形状把它们包围起来
* 为产品功能块使用图标
* 不要让你的用户感到困惑:图标需要有意义，适合文本或动作
```

### Images

```
1. 不同类型的图片:产品照片、讲故事的照片、插图、图案
2. 使用图片来支持你网站的信息和故事:所以只使用相关的图片!
3. 更喜欢原始iamges。如果不可能，使用原始的库存图片(不是通用的!)
4. 试着展示真实的人来触发使用者的情绪
5. 如果需要，裁剪图像，以适应您的信息
6. 实验结合照片，插图和模式
7.  使图像变暗或变亮(完全或部分使用渐变)
8.  定位文本到中立图像区域
9.  将文本放入框中
10. 为了适应高分辨率屏幕，图像尺寸应该是显示尺寸的两倍
11. 压缩图像以降低文件大小和更好的性能
12. 当使用多个图像并排，确保他们有完全相同的尺寸
```
